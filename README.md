# Clone Instructions

```
git clone --recurse-submodules https://github.com/trustworthyrobotics/gpu_sls
```

# Environment Setup

To set up your environment, run:

```
conda env create -f environment.yaml
conda activate gpu_sls
pip install -e .
```
